What is Breaky?

Breaky, the bash script, generates a notification every 20 minutes. That notification serves as a reminder to take a brake from whaterver you are doing on the computer. To stretch, relax your eyes and maybe go refill your glass.

As a fun and interesting bonus, the notification also has three buttons: "Full break", "Short break", "Skip". You can use these to keep track of how often you actually take a break. This data is stored into a database and can be later view on this website.

Which technologies is Breaky built with?

The main part of breaky is a simple bash script, which uses the utilily notify-send.py to create notifications (with buttons!) and then it calls the at command, to schedule another execution of it self in 20 minutes. If you click one of the buttons, the action will be stored to the database.

Interaction with the database is accomplished by calling AWS lambda functions. There are currently two lambda functions, one which stores the data, another which retrieves it.

The database is a mariaDB instance. It has only one table, which stores the prompt date, click date and the resulting action (skip, full, short).

This web app is bulit using Angular. Styles are done using Tailwind CSS. To create the charts, the ngx-charts library is used.
